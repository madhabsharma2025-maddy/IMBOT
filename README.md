# Family Sitcom Scripts (Original Project)

Original animated family-comedy scripting project. Fully separate from any other
project on this machine — no shared files, dependencies, or history.

## Folder structure
```
Family-Sitcom-Scripts/
 ├── characters/     → character bible, personality reference
 ├── episodes/        → episode premise banks (themes, hooks, lesson lines)
 ├── scripts/          → full 30-min beat-by-beat scripts, once expanded
 └── notes/            → misc production notes, running gags tracker, etc.
```

## Status
- Episode scenario bank: 100 premises drafted (episodes/scenario_bank_001-100.md)
- Next: generate batches 101–200+, then expand chosen premises into full scripts.

## Keeping this isolated from other projects
- This folder should live as its own top-level directory — a *sibling* of any other
  project folder, never nested inside one.
- Use its own Git repository (separate `git init`, separate GitHub repo).
- If using VS Code, open this folder in its own window/workspace rather than adding
  it to another project's workspace.
